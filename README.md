

## Building

1. Perform a `git clone` of this repo
1. `cd` into the `simpledoku` directory
1. Run `npm install`
1. Run `npm run watch` in a Terminal Window
1. Run `npm start` in another Terminal Window


## Packaging

1. Run `npm run build`
1. Run `npm run generate`
1. Run `cd app`
1. Run `npm install`
1. Run `npm run package`
